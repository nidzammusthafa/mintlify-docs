## Membuat Template

Template pesan memungkinkan Anda menyimpan pesan yang sering digunakan.

### Langkah Membuat Template

1. Buka menu **Settings** > **Templates**
2. Klik **New Template**
3. Masukkan nama template
4. Tulis konten pesan
5. Gunakan variabel jika perlu: `{nama}`, `{kota}`
6. Simpan template

## Menggunakan Template

Saat membuat blast atau auto-reply:
1. Klik tombol **Load Template**
2. Pilih template yang ingin digunakan
3. Template akan otomatis terisi

## Variabel dalam Template

Gunakan variabel untuk personalisasi:

```
Halo {nama}!

Terima kasih telah menghubungi {perusahaan}.
Kami siap membantu Anda di {kota}.
```

Saat dikirim, variabel akan diganti dengan data dari recipient.
