Ikuti langkah-langkah di bawah ini untuk menginstal dan menyiapkan WazapSuite di perangkat Anda.

## Langkah 1: Instalasi Aplikasi

WazapSuite hadir dalam bentuk aplikasi desktop untuk memastikan keamanan dan stabilitas koneksi WhatsApp Anda.

1.  Unduh installer terbaru dari [Dashboard Member](https://wazapsuite.com/download).
2.  Jalankan file `.exe` (untuk Windows) dan ikuti instruksi instalasi.
3.  Buka aplikasi WazapSuite setelah instalasi selesai.

## Langkah 2: Aktivasi Lisensi

Saat pertama kali dibuka, Anda akan diminta untuk memasukkan Lisensi (License Key).

<Info>
  Lisensi Anda dapat ditemukan di email pembelian atau di dashboard akun Anda.
</Info>

1.  Salin **License Key** Anda.
2.  Tempelkan ke dalam kotak aktivasi di aplikasi.
3.  Klik **Activate**. Aplikasi akan melakukan binding ke perangkat Anda.

## Langkah 3: Menghubungkan Akun WhatsApp

Untuk mulai mengirim pesan, Anda perlu menghubungkan setidaknya satu akun WhatsApp.

1.  Buka menu **Accounts** di sidebar kiri.
2.  Klik tombol **Add Account**.
3.  Sebuah QR Code akan muncul. Buka WhatsApp di HP Anda -> Perangkat Tertaut -> Tautkan Perangkat.
4.  Scan QR Code tersebut.

## Langkah 4: Kirim Blast Pertama Anda

Sekarang Anda siap untuk mengirim pesan pertama!

1.  Masuk ke menu **Blast**.
2.  Klik **New Campaign**.
3.  Impor nomor tujuan (via Excel atau manual).
4.  Tulis pesan Anda dan klik **Send Now**.

<Check>
  Selamat! Anda telah berhasil mengirim kampanye pertama Anda dengan WazapSuite.
</Check>
