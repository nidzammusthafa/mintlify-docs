```typescript
interface MessageTemplate {
  id: string;
  name: string;
  content: string;
  category: string;
  variables: string[]; // Extracted placeholders

  // Media (optional)
  mediaPath?: string;
  mediaType?: "image" | "video" | "document";

  // Statistics
  sentCount: number;
  readCount: number;
  replyCount: number;
  points: number; // Calculated: (read Ã— 2) + (reply Ã— 3)
```

## 10. API Endpoints

| Method | Endpoint                | Description             |
| ------ | ----------------------- | ----------------------- |
| GET    | `/templates`            | List semua template     |
| GET    | `/templates/:id`        | Detail template         |
| POST   | `/templates`            | Buat template baru      |
| PUT    | `/templates/:id`        | Update template         |
| DELETE | `/templates/:id`        | Hapus template          |
| GET    | `/templates/categories` | List kategori           |
| POST   | `/templates/categories` | Buat kategori           |
| GET    | `/templates/top`        | Top templates by points |
| POST   | `/templates/:id/use`    | Record usage            |

---

## 11. Point Update Endpoints

| Method | Endpoint                      | Description             |
| ------ | ----------------------------- | ----------------------- |
| POST   | `/templates/:id/read`         | +2 poin (pesan dibaca)  |
| POST   | `/templates/:id/reply`        | +3 poin (pesan dibalas) |
| POST   | `/templates/:id/reset-points` | Reset poin ke 0         |
