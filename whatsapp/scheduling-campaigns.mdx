## Ikhtisar Penjadwalan

Fitur penjadwalan memungkinkan Anda mengatur kapan pesan blast akan dikirim. Sangat berguna untuk:

- Mengirim di jam kerja saja
- Menyebar kampanye dalam beberapa hari
- Menghindari pengiriman di jam malam

## Cara Mengaktifkan Jadwal

1. Saat membuat kampanye blast, scroll ke bagian **Schedule**
2. Aktifkan toggle **Enable Schedule**
3. Atur waktu mulai dan waktu selesai

## Konfigurasi Time Windows

Anda bisa menentukan jam aktif pengiriman:

```
Contoh:
- 08:00 - 12:00 (Pagi)
- 13:00 - 17:00 (Siang)
- 19:00 - 21:00 (Malam)
```

## Tips Penjadwalan

1. Kirim di jam kerja (08:00 - 20:00)
2. Hindari pengiriman jam 21:00 - 08:00
3. Gunakan beberapa time windows untuk distribusi merata
