## Format Export

WazapSuite mendukung beberapa format export:

| Format | Penggunaan |
|--------|-----------|
| CSV | Untuk import ke sistem lain |
| Excel (.xlsx) | Untuk analisis data manual |

## Menggunakan Data untuk Blast

Data yang di-export bisa langsung digunakan untuk blast:

1. Di halaman **Blast**, klik **New Campaign**
2. Pilih **Import from Scraped Data**
3. Pilih hasil scrape yang ingin digunakan
4. Sistem akan otomatis menggunakan nomor telepon dari data

## Validasi Data

Sebelum blast, disarankan untuk:
- Cek nomor yang duplikat
- Validasi format nomor (mulai dengan 62)
- Filter nomor yang tidak valid
